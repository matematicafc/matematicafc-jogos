# Quiz Material Dourado

Quiz interativo em HTML, CSS e JavaScript para praticar composição e decomposição de números usando material dourado.

## Estrutura

- `index.html`: estrutura das telas do quiz e blocos de anúncio.
- `css/style.css`: estilos visuais, personagens, botões, placar e material dourado.
- `js/script.js`: perguntas, controle do quiz, validação das respostas, áudio e resultado final.

## Como abrir

Abra o arquivo `index.html` diretamente no navegador.
