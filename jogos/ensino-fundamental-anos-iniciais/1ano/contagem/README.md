# Jogo de Contagem

Primeiro jogo do projeto MatematicaFC.

**Público-alvo:** estudantes do 1º ano do Ensino Fundamental.

**Objetivo:** desenvolver habilidades de contagem, reconhecimento de quantidades e associação entre quantidade e numeral, de forma lúdica e interativa.

**Tecnologias utilizadas:** HTML, CSS e JavaScript.

**Autor:** Professor Fernando Cabral.

**Projeto:** MatematicaFC.
